属性选择器引号

属性选择器必须加引号

不正确 

input[type=text] {
    color: red;
}

正确 

input[type='text'] {
    color: red;
}

input[type="text"] {
    color: red;
}

无边框

无边框是 赋值为0还是 none.

选项 描述
样式 none , zero ( 默认 )

none 

.foo {
    border: none;
}

zero 

.foo {
    border: 0;
}

浮点数写法

浮点数 前零 后零 都不写

选项 描述
leading 小数前面的0不可省略
trailing 小数后面的0不可省略
both 小数前后零都需要
none 小数点前后零都不要写( 默认 ).

不正确 

.foo {
  font-size: .5em;  // leading, both
  font-size: 1em;   // trailing, both
  font-size: 1.0em; // none
}

正确 

.foo {
  font-size: 0.5em; // leading, both
  font-size: 1.0em; // trailing, both
  font-size: .5em;  // none
}

层级深度

设置样式的层级深度

选项 描述
depth 从1 到 n 整数 , 3 ( 默认 ).

不正确 (depth 3) 

.foo {
    margin-right: 0;
    .foo-2 {
        color: red;
        .foo-3 {
            width: 100%;
            .foo-4 {
                height: 100%;
            }
        }
    }
}

正确 

.foo {
    margin-right: 0;
    .foo-2 {
        color: red;
        .foo-3 {
            width: 100%;
        }
    }
}

属性重复

不允许有任何重复的属性.

为了兼容性,可以 排除一些属性 不被 lesshint

选项 描述
exclude 排除属性 一个数组. 比如 background-color 就会排除background-color

不正确 

.foo {
    color: red;
    color: blue;
}

正确 

.foo {
    color: red;
}

正确

不正确

正确

16进制颜色

16进制颜色一定要写全6位.不要缩写

选项 描述
style long ( 默认 ), short

long 

.foo {
    color: #000000;
}

short 

.foo {
    color: #000;
}

16进制写法

16进制颜色要小写,便于阅读.

选项 描述
style lowercase ( 默认 ), uppercase

lowercase 

.foo {
    color: #abcdef;
}

uppercase 

.foo {
    color: #ABCDEF;
}

16进制颜色校验

16进制颜色校验是否合法.

不正确 

.foo {
    color: #ab;
}

正确 

.foo {
    color: #abc;
}

id 选择器

不要书写 id 选择器. 因为 id 不容易被复写.

选项 描述
exclude 排除的 ID,用数组表示.

不正确 

#foo {
    color: red;
}

正确 

.foo {
    color: red;
}

important 规则

不允许使用 !important

不正确 

.foo {
    color: red !important;
}

正确 

.foo {
    color: red;
}

import 路径规则

引入文件不能以 _ 开头.

选项 描述
filenameExtension (文件拓展名) false , true ( 默认 )
leadingUnderscore (_) false ( 默认 ), true
exclude 排除的文件数组

不正确 

@import 'foo.less';
@import '_bar';

正确 

@import 'foo';
@import 'bar';

单行最多字符数目

单行不能超过规定的长度

选项 描述
limit 200

不正确 

.foo {
    color: red; // A really long comment that's more than 100 chars...
}

正确 

.foo {
    color: red; // A short comment that's less than 100 chars
}

块后空行

每一个属性块前面都要有一空行.

不正确 

.foo {
    color: red;
}
.bar {
    color: blue;
}

正确 

.foo {
    color: red;
}

.bar {
    color: blue;
}

属性字母排序

确保每个属性都要按照字母排序.

选项 描述
style alpha ( 默认 )

不正确 

.foo {
    color: red;
    border: 0;
}

正确 

.foo {
    border: 0;
    color: red;
}

属性单位

设置那些属性单位可以被使用.

默认情况下可以使用任何单位

选项 描述
valid 允许的单位 (默认情况全部单位都可以)
invalid 不允许使用的单位
properties Object with property names and allowed units (默认为空) 
"propertyUnits": {
    "valid": ["rem", "vw"], // 这些单位可以被任何属性赋值使用.
    "invalid": ["pt"],      //  'pt' 不能被使用.
    "properties": {
        "line-height": []   // line-height 不能使用任何单位.
    }
}

.foo {
    font-size: 1.5rem; // Allowed
    line-height: 30px; // Not allowed
}

元素名前置

选择器不能以元素名开头.

选项 描述
allowWithAttribute false ( 默认 ), true
allowWithClass false ( 默认 ), true
allowWithId false ( 默认 ), true

不正确 

div[foo='bar'] {
    color: red;
}

div.foo {
    color: red;
}

div#foo {
    color: red;
}

正确 

[foo='bar'] {
    color: red;
}

.foo {
    color: red;
}

#foo {
    color: red;
}

选择器命名

选项 描述
disallowUppercase true ( 默认 ), boolean
disallowUnderscore true ( 默认 ), boolean
disallowDash false ( 默认 ), boolean
exclude 字符串数组 
var options = {
    disallowUppercase: true,
    disallowUnderscore: true,
    exclude: ['fooExcluded']
}

正确 

.foo-bar {

}

.fooExcluded {

}

不正确 

.fooBar {

}

.foo_bar {

}

每行写一个属性

每一行只允许写一个属性

选项 描述
单行属性 false ( 默认 ), true

不正确 

.foo {
    color: red; margin-right: 10px;
}

.bar { color: red; }

正确 

.foo {
    color: red;
    margin-right: 10px;
}

.bar {
    color: red;
}

正确 allowSingleLineRules: true 

.bar {
    color: red;
}

.bar { color: red; }

单行单选择器

每一行只能选择一个元素.

不正确 

.foo, .bar {
    color: red;
    }

正确 

.foo,
.bar {
    color: red;
}

分号后空格

分号后面一个空格.

选项 描述
style no_space , one_space ( 默认 )

no_space 

.foo {
    margin:0;
}

one_space 

.foo {
    margin: 0;
}

属性值后面空格

属性后不加空格

选项 描述
style no_space ( 默认 ), one_space

no_space 

.foo {
    margin: 0;
}

one_space 

.foo {
    margin : 0;
}

属性值后面空格

属性值后面不加空格

选项 描述
style no_space ( 默认 ), one_space

no_space 

.foo {
    margin: 0;
}

one_space 

.foo {
    margin: 0 ;
}

逗号后面空格

逗号后面加空格

选项 描述
allowNewline false ( 默认 ), boolean
style after ( 默认 ), before , both , none

after 

.foo {
    color: rgb(255, 255, 255);
}

before 

.foo {
    color: rgb(255 ,255 ,255);
}

both 

.foo {
    color: rgb(255 , 255 , 255);
}

none 

.foo {
    color: rgb(255,255,255);
}

allowNewline : true 

.foo {
  font:
    14px,
    Roboto,
    #000;
}

运算符加号前后空格

定义 运算符 ( + , - , * , / , and = ) 前后空格怎么加更加易读.默认前后都加一个空格.

选项 描述
style both ( 默认 ), none

both 

.foo {
    height: calc(10px + 10px);
}

none 

.foo {
    height: calc(10px+10px);
}

花括号开始前面空格

花括号开始前面加一个空格.

选项 描述
style no_space , one_space ( 默认 ), new_line

no_space 

.foo{
    color: red;
}

one_space 

.foo {
    color: red;
}

new_line 

.foo
{
    color: red;
}

小括号内侧空格

小括号内部不需要空格默认.

选项 描述
style no_space ( 默认 ), one_space

no_space 

.foo {
    color: rgb(255, 255, 255);
}

one_space 

.foo {
    color: rgb( 255, 255, 255 );
}

字符串引号

字符串默认使用单引号.

选项 描述
style double , single ( 默认 )

不正确 

.foo {
    content: "Hello world";
}

正确 

.foo {
    content: 'Hello world';
}

末尾分号

需要末尾分号.

不正确 

.foo {
    color: red
}

正确 

.foo {
    color: red;
}

url 格式

选项 描述
style absolute , relative ( 默认 )

不正确 

.foo {
    background-image: url('http://example.com/img/image.jpg');
}

正确 

.foo {
    background-image: url('img/image.jpg');
}

通配符选择器

不能使用通配符选择器

不正确 

* {
    color: red;
}

.foo * {
    color: red;
}

正确 

.foo {
    color: red;
}

.foo .bar {
    color: red;
}

url 引号

默认不要引号.

不正确 

.foo {
    background-image: url('img/image.jpg');
}

正确 

.foo {
    background-image: url(img/image.jpg);
}

变量的值

属性的值是否使用变量.

选项 描述
always 总是使用变量的.
never 从不使用变量的.

always ['color'] 正确 

.foo {
    color: @success;
}

always ['color'] 不正确 

.foo {
    color: #333;
}

never ['color'] 正确 

.foo {
    color: #333;
}

never ['color'] 不正确 

.foo {
    color: @success;
}

0值单位

值为的0的属性,不写单位.

选项 描述
style no_unit ( 默认 ) , keep_unit

no_unit 

.foo {
    margin-right: 0;
}

keep_unit 

.foo {
    margin-right: 0px;
}
